1. 収集する情報

本アプリは、次の情報を取り扱います。

  • メールアドレス（メール登録を選択した場合のみ）
  • 参戦記録（各試合の参戦ステータス、交通手段・宿泊のメモ）

「この端末だけで使う」を選択した場合、本アプリは外部のサーバーと一切通信しません。記録はお使いの端末内にのみ保存され、運営者は何も受け取りません。同期に必要なプログラムは、メール登録を選んだときに初めて読み込まれます。

2. 利用目的

  • メールアドレス：本人確認（ログインリンクの送信）、および記録を端末間で同期するため
  • 参戦記録：本人に表示・同期するため

広告配信や、マーケティング目的での利用は行いません。

3. 第三者への提供

運営者は、取得した情報を第三者に販売・提供しません。ただし、以下のサービスを利用しています。

  • GitHub Pages（本アプリの配信）— 利用モードにかかわらず、ページの表示時に通信が発生します
  • Supabase（データベース・認証）— メール登録を選んだ場合のみ
  • esm.sh（同期プログラムの配信）— メール登録を選んだ場合のみ

法令に基づく開示要請があった場合は、この限りではありません。

4. 他の利用者からの見え方

あなたの参戦記録やメールアドレスが、他の利用者に個人単位で公開されることはありません。データベース側の権限設定（行レベルセキュリティ）により、各利用者は自分の記録のみにアクセスできます。

5. データの削除

アプリ内の「アカウント」画面から、いつでもアカウントと記録の全てを削除できます。削除は即時に実行され、元に戻すことはできません。

6. Cookie・アクセス解析

本アプリはログイン状態の保持のためにブラウザのローカルストレージを使用します。アクセス解析ツールや広告目的の Cookie は使用していません。

7. お問い合わせ

個人情報の取り扱いに関するお問い合わせ・開示請求は、 までお願いします。
